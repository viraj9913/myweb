Self Project on Machine Learning

Forecasting future GDP of the UK


This was the project which I performed for understading the forecasting topic in Machine Learning concept. In addition, the project introduced with different forecasting models such as: ARIMA & SARIMAX. Whereas, VAR model was also executed to compare its response with other forecasting models. Linear Regression helped in finding the relationship between independent and dependent feature. Both models ARIMAX & SARIMAX, showed its signifance for prediction.

SARIMAX model offered seasonality which becomes essential, where there is a specific pattern in the data. These patterns maybe a certain time period of the year, when we observe the rise or fall of value. With this functionality, our model can understand the trend exceptionally well for each year, which makes it idle for seasonal data.
On the other hand, ARIMA model is quite straight-forward compared to SARIMAX. It doesn't provide seasonality, meaning model fails to understand the pattern when there is rise of value for certain time period of the year. Instead, ARIMA model is more focused on trend which has either increasing, decreeasing or stationary pattern. Since our dataset deals with the GDP of the UK, the data shows the rise of GDP for each year. This model was also superior for this project. Whereas, VAR model was also employed which behaves similar to ARIMA & SARIMAX model.

In order to use these models, data must be stationary. We can perform adfuller test, to ensure whether the data is stationary or not. If there is no stationary, we perform differencing on the column which is not stationary. This makes the dataset stationary and ready to use by model. Further, we plot ACF and PACF chart to obtain the value of 'p', 'q' which are the parameters used in these models. 'd' is also another parameter which is differencing. We assign the value of 'd' based on the number of times differencing performed. Hence, selection of the correct value for these parameters, makes the model prediction significantly accurate. German Traffic Signal Recognition


The German Traffic Signal Recognition project was performed during my Master's Course. The aim of the project was to build a machine learning model which can detect the German Traffic Signal signs and predict the image labels. To begin with, EDA (Exploratory Data Analysis) was done to highlight insights from the data. The train data was provided which contains more than 39,000 images and test data holds more than 11,000 images. The model was build using CNN (Convoluional Neural Network). It has the hidden layer aka (Convoluional layer) which detect the pattern in the image. This can be specified by number of filters. The deeper the network goes, more sophisticated network becomes. These filters forms a small matrix, where we can specify number of rows and column. In this case, we have use 3x3 matrix, which obtains 3x3 pixels from the image. This process continues until it covers an entire image. Furthermore, we capture this 3x3 pixel and store the dot product, which will be passed as the next output layer.
The link for this project is attached to demonstrate the step by step approach, which is accessible on GitHub.
